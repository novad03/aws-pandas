AWS Data Wrangler
=============================================

*Utility belt to handle data on AWS.*

Use Cases
---------

* Pandas -> Parquet (S3)
* Pandas -> CSV (S3)
* Pandas -> Glue Catalog
* Pandas -> Athena
* Pandas -> Redshift
* CSV (S3) -> Pandas (One shot or Batching)
* Athena -> Pandas (One shot or Batching)
* PySpark -> Redshift
* Delete S3 objects (parallel :rocket:)
* Encrypt S3 data with KMS keys
